Guide d'installation de CalculatorGWT

Instructions détaillées pour installer et utiliser votre calculatrice scientifique

Sommaire

  • Prérequis
  • Téléchargement
  • Extraction
  • Lancement
  • Dépannage

1. Prérequis système

Avant de procéder à l'installation, assurez-vous que votre système répond aux exigences suivantes :

Navigateurs supportés

  • Google Chrome 80+
  • Mozilla Firefox 75+
  • Safari 13+
  • Microsoft Edge 80+

Configuration minimale

  • JavaScript activé
  • Au moins 50 MB d'espace disque libre
  • Résolution d'écran 1024x768 minimum

2. Téléchargement

Procédez au téléchargement du package complet :

  1. Rendez-vous sur la page de téléchargement
  2. Cliquez sur le bouton "Télécharger CalculatorGWT.zip"
  3. Enregistrez le fichier dans un emplacement facilement accessible (Bureau ou Documents)
Note : Le fichier ZIP pèse environ 15 MB selon la vitesse de votre connexion.

3. Extraction des fichiers

Extrayez le contenu du fichier ZIP :

Sur Windows :

  1. Faites un clic droit sur le fichier CalculatorGWT.zip
  2. Sélectionnez "Extraire tout..."
  3. Choisissez le dossier de destination (ex: C:\Program Files\CalculatorGWT\)
  4. Cliquez sur "Extraire"

Sur macOS :

  1. Double-cliquez sur le fichier CalculatorGWT.zip
  2. Le contenu sera automatiquement extrait dans le même dossier

Sur Linux :

unzip CalculatorGWT.zip -d ~/CalculatorGWT/

Structure des fichiers après extraction :

CalculatorGWT/
├── CalculatorGWT.html      # Fichier principal à ouvrir
├── CalculatorGWT.css       # Feuilles de style
├── CalculatorGWT.js        # Script principal
├── clear.cache.gif         # Fichiers de cache
├── historique.txt          # Historique des calculs
└── ... autres fichiers

4. Lancement de l'application

Pour démarrer CalculatorGWT :

  1. Ouvrez le dossier où vous avez extrait les fichiers
  2. Double-cliquez sur le fichier CalculatorGWT.html
  3. L'application s'ouvrira dans votre navigateur par défaut
Astuce : Vous pouvez créer un raccourci sur votre bureau vers le fichier CalculatorGWT.html pour un accès rapide.

Premier lancement :

Lors du premier lancement, l'application peut mettre quelques secondes à se charger car elle initialise tous les modules GWT. Les lancements suivants seront plus rapides.

5. Dépannage

L'application ne se charge pas

Solution : Vérifiez que JavaScript est activé dans votre navigateur.

Erreur de sécurité

Solution : Si vous voyez une alerte de sécurité, autorisez le chargement du contenu. L'application s'exécute localement et ne nécessite pas de connexion Internet.

Interface non responsive

Solution : Redimensionnez la fenêtre de votre navigateur ou actualisez la page.

Fonctionnalités manquantes

Solution : Assurez-vous que tous les fichiers ont été correctement extraits et se trouvent dans le même dossier.

Si vous rencontrez d'autres problèmes, consultez notre page de contact pour obtenir de l'aide.
